# AI-Powered Vibe Calendar

The goal of this project is to make a highly customizable calendar tool, that combines commitments that users have to make, and tasks they have to complete.
The program should automatically generate a calendar based on the user's schedule, tasks, and preferences.
This project was originally designed for mobile, but since it was made on Flutter, I have published the web build since it is easier to deploy for HC demos.

Demo Link: https://ai-dev-cs.github.io/calendar-web/
